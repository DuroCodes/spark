## CommandPlugin

Represents a plugin that is ran upon the command being used.
Good for guards, reusability.

| name        | string                                                                     | N/A   | -           |
| run         | (options: CommandPluginOptionsDefs\[T]) => Awaitable\<Result\<void, void>> | N/A   | -           |
